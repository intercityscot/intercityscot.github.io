A change has occurred on the service that hosts The Inter-city Scot, disabling the form submission capability.

To add a listing to The Inter-city Scot, please

  1. Look at one of the examples below,
  2. Create (in an email message) a comparable listing for your listing
  3. Send the listing in an email message to webmaster@intercityscot.org.

To delete a listing from The Inter-city Scot, please

  1. Copy the listing you would like to delete
  2. Paste the listing into an email message
  3. Send an email message to webmaster@intercityscot.org.

To update a listing in The Inter-city Scot, please

  1. Copy the listing you would like to update
  2. Paste the listing in an email message twice.
  3. Edit the second copy of the listing to show how you would like the listing to appear (preferably using red text to show new or changed text).
  4. Send the email message to webmaster@intercityscot.org.

Example:

Original:

La Grande Scottish Country Dancers [Feb 2018]
Tim Harrison (Teacher)
email: tim.harrison@domain.com

Revised:

La Grande Scottish Country Dancers [May 2019]
Tim Harrison (Teacher)
email: tharrison@bogus.com
